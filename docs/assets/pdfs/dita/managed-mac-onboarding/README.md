# DITA Sample: Managed Mac Onboarding Guide

## Summary
This project adapts an end-user macOS onboarding guide into a modular DITA documentation set. It demonstrates structured authoring by separating explanatory, procedural, reference, and troubleshooting content into distinct DITA topic types and organizing them with a DITA map.

## Audience
- New hires receiving a company-managed Mac
- Non-technical employees
- Contractors working in an enterprise environment
- Hiring managers reviewing structured writing samples
- Documentation teams using DITA or topic-based authoring

## Documentation Goal
The goal of this sample is to help users complete initial managed Mac setup with minimal IT assistance while demonstrating foundational DITA skills, including topic-based authoring, content modularity, and map-based organization.

## DITA Concepts Demonstrated
- Concept topic
- Task topic
- Reference topic
- Troubleshooting topic
- DITA map
- Topic-based authoring
- Structured XML content organization

## Tools / Methods
- DITA XML
- GitHub
- Structured authoring principles
- Topic-based writing
- Content modularization

## What I Learned
This sample helped me practice restructuring a linear onboarding guide into modular DITA topics. It reinforced the distinction between concept, task, reference, and troubleshooting content and showed how a documentation set can be organized through a DITA map.

## Notes
This project was created as a foundational DITA writing sample for a technical writing portfolio. The scope was intentionally kept focused to demonstrate core DITA concepts clearly.
