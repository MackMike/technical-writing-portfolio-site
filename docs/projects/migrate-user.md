# Migrate User Administrative Tool Documentation

This sample documents an AppleScript-based administrative tool for restoring ownership and group permissions to a migrated user's home folder after a Mac has been reimaged. The project includes a polished administrator README and a revised script with clearer prompts, improved warnings, maintainable comments, and more administrator-focused messaging.
